# 🏨 Pink City Hotel

A modern **Hotel & Banquet Booking Web Application** built using **React, Vite, Firebase, Node.js, and Supabase**. The platform allows users to browse hotel rooms, banquet halls, view galleries, make bookings, and contact the hotel. It also includes an admin dashboard for managing hotel content.

---

## 🚀 Features

### 👤 User Features

- Browse hotel rooms
- View banquet halls
- Room details page
- Hall details page
- Hotel gallery
- Contact page
- About page
- Online booking
- User Login & Registration
- Responsive UI
- Google Map integration

---

### 🔐 Admin Features

- Admin Login
- Dashboard
- Upload Rooms
- Upload Halls
- Manage Hotel Data
- Firebase Authentication
- Image Upload Support

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Pink-City-Hotel.git
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

### Backend

```bash
cd backend
npm install
node server.js
```

or

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

JWT_SECRET=your_secret

SUPABASE_URL=your_url
SUPABASE_KEY=your_key

FIREBASE_API_KEY=your_key

CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret

EMAILJS_PUBLIC_KEY=your_key

RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```
